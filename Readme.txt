Quick Start Guide

Step 1 Navigate to the Project Directory
Find the CantCodeDenSleep folder where you’ve stored the project.

cd CantCodeDenSleep

Step 2 Open the Terminal
Launch the terminal
For Windows Command Prompt or PowerShell
For Mac Terminal
For VS Code Use the terminal from the top bar

Step 3 Install Dependencies
Run the following command to install the necessary packages

npm install

Step 4 Start the Project
Once the installation is done run

npm start

The simulation will now be live and you can start solving transportation crises 
Happy Destroying (✘) Simulating (✓)!
